MediReady

MediReady discretely stores up to three different medications of any type, from pills to liquids. Custom-made pressure sensors allow container to monitor if the user has missed doses and lights inside notify users of missed doses. An internal calender with information on missed doses can be downloaded by the user to later bring to doctor's appointments, eliminating the work behind manually tracking medication schedules. A matching portable clip stores up to 3 different medications and through bluetooth communicates missed doses to the homebase.

Vibrance

The Vibrance Monkey is the current iteration of the system with a human-based design, providing students aged 3-5 in the Deaf community a fun way to experience music and rhythm. This is done by transmitting the music as vibrations through the wearable backpack.

Dance Party in a Box

The Dance Party in a Box is a portable sound mixer that allows the user to create looping synth music with a reactive light display. A connected display shows how the synth affects waveforms as the user creates music. By using this synth the user can visual how different waveforms sound and easily learn the physics behind soundwaves.
